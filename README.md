# Room Database App

A simple Android app demonstrating **Room Database** functionality using Kotlin.  
This app allows storing, retrieving, and displaying data using Room ORM.

---

## Features

- Add, update, and delete data in the database
- View stored data in a RecyclerView
- Modern and simple UI
- Fully functional offline storage with Room
